Меню воркшопов

В дружеской обстановке пройдут следующие воркшопы:

Javascripting

Основы JavaScript с нуля. Опыт программирования не требуется.

Bug clinic

Современные инструменты и техники, которые улучшат ваши навыки отладки кода.

Test anything

Как тестировать свой собственный код оптимальными способами.

про nodeschool

Это оpen source воркшопы, которые обучают веб-разработке.

Сообщество проводит встречи в Киеве, чтобы помогать проходить воркшопы и обмениваться опытом.

Для всех желающих программировать

Мы хотим, чтобы все узнали, как здорово программировать, и рады всем желающим посетить встречу: тем, кто хочет научиться, и тем, кто хочет учить!

Для обучения программированию

Присоединяйтесь и изучите основы Node.js с нами. Если у вас есть опыть в JavaScript - это плюс, но если нет - это не страшно.

Для самостоятельного изучения

Воркшоп - это набор заданий для индивидуального прохождения. Нет длинных лекций. Все задания под лицензией open source и доступны на nodeschool.oi

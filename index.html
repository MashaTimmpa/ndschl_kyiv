Меню воркшопів

В дружній атмосфері пройдуть наступні воршкопи:

Javascripting

Основи JavaScript для початківців. Не потребує досвіду програмування.

Bug clinic

Сучасні інструменти та техніки, які покращать ваші навички для налагодження коду.

Test anything

Як тестувати свій власний код оптимальними способами.

про nodeschool

Це оpen source воркшопи, які навчають веб-розробці.

Спільнота проводить зустрічі в Києві для обміну досвідом та для допомоги у проходження воркшопів.

Для всіх бажаючих програмувати

Ми хочему, щоб усі знали, що програмувати - надзвичайно чудово, і раді усім бажаючим відвідати зустріч: і тим, хто хоче вчитится, і тим, хто бажає навчати!

Для навчання програмуванню

Приєнуйтеся та вивчайте основи Node.js з нами. Якщо ви маєте досвід з JavaScript - це плюс, та якщо й ні - це не страшно.

Для самостійного вивчення

Воркшоп - це набір завдань для індивідуального проходження. Нема тривалих лекцій. Всі завданя під ліцензією open source та доступні на nodeschool.oi
